基本のテンプレート 

#include <bits/stdc++.h>
using namespace std;

int main() {
    // ここから実行される！
    /* この中に命令を書いていきます */
    
    return 0;
}

#include <bits/stdc++.h>
C++の便利な機能を全部詰め合わせた「お得セット」です。これ一つでOK！

main() 関数
プログラムは必ずこの中にある命令を上から順に実行します。
（Rustだと fn main()、Javaだと public static void main みたいな役割です）

基本の計算 

cout << 10 + 20 << endl; // 30 (和)
cout << 10 - 5 << endl;  // 5  (差)
cout << 5 * 2 << endl;   // 10 (積)
cout << 20 / 3 << endl;  // 6  (商)
cout << 20 % 3 << endl;  // 2  (余り)
cout << (20 + 5) * 2 << endl;//(括弧)
⚠️ 割り算の注意
整数同士の割り算（20 / 3）は、小数点以下が切り捨てられて整数になります。

変数（データの箱）

値を入れておくための「箱」です。入れるものによって「型」を使い分けます。

型 説明 例
int 整数 10, -500
long long 大きな整数 10の18乗くらいまで
double 小数 3.14, -0.5
string 文字列 "Hello" 
int i = 100;
double d = 1.5;
string s = "This is IQ";

cout << i << endl;
cout << d << endl;
cout << s << endl;

入力：cin（最重要）

キーボードから入力を受け取る方法です。競プロはこれができないと始まりません！

int a, b;
// 2つの整数をスペース区切りで入力
cin >> a >> b; 

// 入力された値を計算して出力
cout << "合計は: " << a + b << endl;

※ cin は「シィ・イン」、cout は「シィ・アウト」と呼びます。矢印の向きに注目！
