ICPC向け：C言語の文字列とchar型

C言語には「文字列型」が存在しない！？
文字列の正体を理解して、ICPCの問題を解き進めよう。

★ 文字列は配列の仲間である ★

1. char型とは何か

文字列を扱う前に、まずは文字について説明します。
char 型は、1文字だけを保存するための型です。 

#include <stdio.h>

int main(void)
{
    char c = 'A';
    printf("%c\n", c); // A が出力される

    return 0;
}

■ char型は「実は整数」

文字はコンピュータの内部では数値として保存されています（ASCIIコード）。数値として出力（%d）すると、その正体がわかります。 

#include <stdio.h>

int main(void)
{
    char c = 'A';
    printf("%d\n", c); // 65 が出力される

    return 0;
}

2. C言語には「文字列型」がない

Pythonの s = "hello" や、C++の string s = "hello"; のように、C言語には文字列専用の型はありません。
C言語において文字列とは、「char の配列」として扱われます。一文字ずつ順番に配列に入っているイメージです。

■ 文字列の正体と終端文字（\0）

例えば char s[] = "hello"; というコードは、内部的には次のような文字の配列として扱われています。 

char s[] = {
    'h',
    'e',
    'l',
    'l',
    'o',
    '\0'
};
インデックス 0 1 2 3 4 5
文字 'h' 'e' 'l' 'l' 'o' '\0'

⚠️ \0（ヌル文字）とは？

文字列の終端を示す特殊な文字です。これがないと、コンピュータは「配列のどこまでが文字列なのか」を判断できず、関係ないメモリまで読み込んで文字化けやエラーを引き起こします。

3. 文字列の入力と長さの取得

■ 文字列の入力

文字列を入力として受け取る場合、scanf には %s を使います。
注意点として、配列を渡す場合は & は不要です。 

#include <stdio.h>

int main(void)
{
    char s[100];
    scanf("%s", s);    // ＆はつけない！
    printf("%s\n", s);

    return 0;
}

■ strlen (文字列の長さ)

文字列の長さを求めたい場合は、string.h をインクルードして strlen() 関数を使います。
※ 終端文字（\0）は長さに含まれません。 

#include <stdio.h>
#include <string.h>

int main(void)
{
    char s[] = "hello";
    printf("%zu\n", strlen(s)); // 5 が出力される

    return 0;
}
