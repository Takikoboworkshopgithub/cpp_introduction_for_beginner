A問題：最強のカードを見つけろ

変わった順序規則を整理し、条件分岐へ落とし込む
競技プログラミングの第一歩、条件の優先順位を見極めよ！

★ 複雑なルールをシンプルな条件分岐に解きほぐせ ★

0. 問題文と解釈・翻訳

問題の原文（AOJ公式）はこちら： AOJで問題文を開く ➔

トランプの強さを普通の順序ではなく、ちょっと変わった「大老二順」というルールで並び替えたときに、手札の中で一番強いカードがどれかを答える問題です。

【強さの順番】

2 (一番強い) ＞ 1 (A) ＞ 13 (K) ＞ 12 (Q) ＞ 11 (J) ＞ 10 ＞ …… ＞ 3 (一番弱い)

1. 法則性・実装方法

このルールをよーく見ると、実は次の3つのシンプルな事実（優先順位）に気づけます。

① 手札に「2」があれば、それが絶対に一番強い。

② 「2」がなくて「1」があれば、それが次に強い。

③ 「2」も「1」もなければ、残りの数字（3〜13）の中で一番大きい数字が強い。

今回は、この優先順位をそのまま素直にプログラムに落とし込んだ2つの素敵な解法を紹介します。どちらの方法も、初心者の方にとって非常に分かりやすいアプローチになっています！

パターン1：優先順位で上からチェックする作戦

1つ目は、先ほどの「3つの事実」を上から順番に if 文でチェックしていく、とても直感的なコードです。

  • count(C.begin(), C.end(), 2) を使って、手札の中に「2」が1枚でもあるか数えます。あれば即座に 2 を出力します。
  • なければ、次に「1」があるか数えます。あれば 1 を出力します。
  • どちらもなければ、残ったカード（3〜13）の中で一番大きい数字が最強になるので、*max_element を使って手札の最大値を探して出力します。 
#include <bits/stdc++.h>
using namespace std;

int main() {
    while (true) {
        int N;
        cin >> N;
        if (N == 0) break;

        vector<int> C(N);
        for (int i = 0; i < N; i++) cin >> C[i];

        // 優先順位①：手札に「2」が含まれるか
        if (count(C.begin(), C.end(), 2)) {
            cout << 2 << '\n';
        }
        // 優先順位②：手札に「1」が含まれるか
        else if (count(C.begin(), C.end(), 1)) {
            cout << 1 << '\n';
        }
        // 優先順位③：それ以外なら単純な最大値
        else {
            cout << *max_element(C.begin(), C.end()) << '\n';
        }
    }
    return 0;
}

パターン2：強い順に「持ってる？」と聞いていく作戦

2つ目は、あらかじめ「強いカードの順番」のリストを用意しておき、そのリストの上から順番にチェックしていくスマートなコードです。

  • 最初に、大老二順の強い並び（2, 1, 13, 12 ... 3）をそのまま row という名前の配列に用意しておきます。
  • 入ってきた手札を、連想配列（map）を使って「どの数字が何枚あるか」をメモ（カウント）しておきます。
  • あとは、強い順リスト（row）の先頭から順番に「このカード、手札に持ってる？」と map に聞いていきます。持っている（枚数が0より大きい）カードが最初に見つかった時点で、それが「手札の中の最強カード」に決定するので、それを出力して処理を終了（break）します。 
#include <bits/stdc++.h>
using namespace std;

void solve(int n) {
    // 大老二ルールに基づいた「強い順」のリストを定義
    vector<int> row = {2, 1, 13, 12, 11, 10, 9, 8, 7, 6, 5, 4, 3};
    map<int, int> count;
    
    while(n--) {
        int a; cin >> a;
        count[a]++; // 手札にあるカードの枚数を記録
    }
    
    // リストの先頭（最も強いカード）から順番に走査
    for (int i = 0; i < 13; i++) {
        if (count[row[i]] > 0) {
            cout << row[i] << '\n';
            break; // 最初に見つかった強いカードを出力して終了
        }
    }
    return;
}

int main() {
    while (true) {
        int n; cin >> n;
        if (n == 0) break;
        solve(n);
    }
    return 0;
}
