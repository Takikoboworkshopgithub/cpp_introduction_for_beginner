第00講：コマンドラインの使い方

競技プログラミングの第一歩。
マウスを捨てて、キーボードでPCを操る感覚を掴もう。

★ 黒い画面のコマンドライン ★

必須コマンド 7選

コマンド フルスペル 説明
ls list 現在のフォルダ内のファイルを表示
mkdir make directory 新しいフォルダを作成
cd change directory フォルダへ移動（cd ..で戻る）
cp copy ファイルをコピー（複製）する
mv move 移動・名前変更（上書き注意！）
g++ GNU C++ Compiler ソースコードを機械語に変換
./a.out execute current コンパイルしたプログラムを実行

コマンド実行シミュレーション（📍 ＝ 現在地）

STEP 0：現状確認

$ ls
📍 /home/user/
└── (空っぽ)

STEP 1：フォルダ作成

$ mkdir folder
📍 /home/user/
└── folder/ (NEW!)

STEP 2：フォルダへ潜る

$ cd folder
/home/user/
└── 📍 folder/ (移動完了)

STEP 3：エディタ起動

$ code sample.cpp
📍 folder/
└── sample.cpp (保存完了)

STEP 4：コンパイル

$ g++ sample.cpp
📍 folder/
├── sample.cpp
└── a.out (生成成功)

STEP 5：プログラム実行

$ ./a.out
Hello World!

STEP 6：整理用のフォルダ作成

$ mkdir folder2
📍 folder/
├── a.out
├── sample.cpp
└── folder2/ (NEW!)

STEP 7：ファイル移動

$ mv sample.cpp folder2/
📍 folder/
├── a.out
└── folder2/
      └── sample.cpp (移動完了)

STEP 8：中身を確認

$ ls
a.out folder2

STEP 9：深い階層へ移動

$ cd folder2
folder/
└── 📍 folder2/
      └── sample.cpp

STEP 10：最終確認

$ ls
sample.cpp

💡 作業を効率化するTips

  • Tabキー： 入力中のフォルダ名を自動で補完します。
  • ↑キー： 一つ前に打ったコマンドを呼び出せます。
  • Ctrl + C： 無限ループなどでプログラムを止めたい時に使います。
  • pwd： 今いる場所の「フルパス」を表示します。
