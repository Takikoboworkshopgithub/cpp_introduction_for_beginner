★ 条件分岐させてみよう ★

1. if構文とスコープ

C++の条件分岐は { } でブロックを作ります。1行でも省略しないのが競プロの安全策です。 

#include <iostream>
using namespace std;

int main() {
    int score = 85;

    if (score >= 80) {
        cout << "合格です！" << endl;
    } else if (score >= 60) {
        cout << "追試です。" << endl;
    } else {
        cout << "不合格です。" << endl;
    }
}
出力：
合格です！

2. 比較演算子と論理演算子

「かつ(&&)」「または(||)」「否定(!)」を使いこなしましょう。 

#include <iostream>
using namespace std;

int main() {
    int age = 20;
    bool has_ticket = true;

    if (age >= 18 && has_ticket) {
        cout << "入場可能です。" << endl;
    }

    if (age < 18 || !has_ticket) {
        cout << "入場できません。" << endl;
    }
}
出力：
入場可能です。

3. C++の「真偽値」の正体（整数評価）

C++では 「0なら偽、0以外ならすべて真」 と判定されます。

#include <iostream>
using namespace std;

int main() {
    int n = 10;

    if (n) {
        cout << "nは0ではありません。" << endl;
    }

    if (n % 2) {
        cout << "nは奇数です。" << endl;
    } else {
        cout << "nは偶数です。" << endl;
    }
}
出力：
nは0ではありません。
nは偶数です。

4. 爆速入出力の「おまじない」

大量のデータをさばくICPCやAtCoderでは必須の高速化設定です。

int main() {
    // 爆速化のおまじない（mainの先頭に書く）
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    // 処理...
}

※ これを書くだけで実行時間が数分の一になることがあります。
