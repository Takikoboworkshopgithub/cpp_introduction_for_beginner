第06講：STL(1) 便利ツールと並び替え

C++の最強武器庫「STL」を使いこなす
ソート、ペア、そして「イテレータ」の正体

★ 自分で書かずに、ライブラリに任せよ ★

0. 本講のゴール

  • sort と reverse でデータを自由自在に並び替える
  • min / max / abs / swap などの便利関数を覚える
  • 「イテレータ」を「配列を指す矢印」としてイメージできる
  • pair を使って2つの関連するデータをまとめて管理する

1. 基本の便利ツール

数値を扱う際によく使う、短いけれど強力な関数たちです。

int a = -10, b = 20, c = 30;

cout << abs(a) << endl;    // 絶対値: 10
cout << min(b, c) << endl; // 小さい方: 20
cout << max(b, c) << endl; // 大きい方: 30

swap(b, c);                // bとcの中身を入れ替える
cout << b << " " << c << endl; // 30 20

2. ソートと反転（イテレータの理解）

配列全体を操作するとき、v.begin()（先頭の矢印）と v.end()（末尾の次を指す矢印）を指定します。 

vector<int> v = {3, 1, 4, 1, 5, 9};

// 小さい順に並び替え
sort(v.begin(), v.end()); 
// 結果: {1, 1, 3, 4, 5, 9}

// 配列の前後をひっくり返す
reverse(v.begin(), v.end());
// 結果: {9, 5, 4, 3, 1, 1}

💡 豆知識：イテレータのイメージ

イテレータは「配列の特定の場所を指す矢印」です。

  • v.begin()：0番目の要素を指す矢印
  • v.end()：「最後の要素の、一つ後ろ」を指す矢印

※画像：[./images/iterator.png]（矢印が配列の端を指している図）

3. 二つの値をセットにする（pair）

「出席番号と点数」「x座標とy座標」など、セットで管理したいデータに使います。

pair<string, int> p = {"Takiguchi", 100};

cout << p.first << endl;  // 1つ目の値: Takiguchi
cout << p.second << endl; // 2つ目の値: 100

// vectorに入れると超強力
vector<pair<int, int>> points;
points.push_back({10, 20});
points.push_back({5, 30});

// pairのvectorをソートすると、まず first で比較、同じなら second で比較される！
sort(points.begin(), points.end());

💻 ワークショップ

演習：座標ソート

N個の座標 (x, y) が与えられます。これを「x座標が小さい順」に並び替えて出力してください。
ただし、x座標が同じ場合は「y座標が小さい順」に並び替えてください。

入力例：
3
10 20
5 10
10 15

出力例：
5 10
10 15
10 20

  • pair は複数データをまとめてソートするのに最強
