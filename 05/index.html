第05講：配列と文字列

データの集まりを扱う → 1次元・2次元配列と文字列操作
変数の「箱」を連結し、大量のデータを一括管理する重要回。

★ 大量のデータを一網打尽にせよ ★

0. 本講のゴール

  • 大量のデータを「配列」で一括管理する
  • 0番目から始まるという鉄則を叩き込む
  • 文字列（string）を配列のように操作する
  • 2次元配列を使って表やグリッドデータを表現する

1. 一次元配列（vector）

クラス10人の点数を保存するようなコードを書きたい場合、今までの知識だとどうなるでしょうか？ 

int main(){
    /* 10人分の変数を確保（大変すぎる…） */
    int a, b, c, d, e, f, g, h, i, j;
    a = 4;
    b = 1;
    c = 2;
    // ...
}

1次元配列は、箱が横に連なっているようなものです。intの箱がたくさんある状態を考えてみましょう。

■ 宣言の仕方と解決策

#include <bits/stdc++.h>
using namespace std;

int main(){
    vector<int> test(10); // testが変数にあたる。10個の箱を用意。
    test[0] = 4;
    test[1] = 1;
    test[2] = 2;
    // ...
}

⚠️ 超重要注意ポイント

  • 配列は「0番目」から始まります。
    最初のうちは1番目と勘違いしてバグらせがちです。
  • 存在しない場所を指定するとバグる（エラーで落ちる）。
int main(){
    vector<int> a(10);// 10個の箱を宣言
    cout << a[10] << endl; // 配列は 0〜9の10個 しかないので、a[10]は11番目を指すことになりエラー！
}

■ よくある書き方（N個の入力と合計）

競プロで最も頻出する「N個のデータを受け取って合計を求める」テンプレです。

#include <bits/stdc++.h>
using namespace std;

int main(){
    int sum = 0;
    int N;
    cin >> N;
    
    vector<int> a(N);
    // N個のデータを受け取る
    for(int i = 0; i < N; i++){
        cin >> a[i];
    }
    
    // 足し合わせる
    for(int i = 0; i < N; i++){
        sum = sum + a[i];
        // sum += a[i]; でも可
    }
    cout << sum << endl;
}

2. 二次元配列

クラス10人の「国・英・数」の点数を保存したい場合。

int main(){
    // 別々に配列を作ると管理が面倒…
    vector<int> Japanese(10);
    vector<int> English(10);
    vector<int> Mathmatics(10);
}

→ 理想は、大きな配列1個の中に、さらに配列を入れる構造です。

■ 二次元配列の書き方

#include <bits/stdc++.h>
using namespace std;

int main(){
    /* 3つの大きな箱の中に vector<int>(サイズ10) があるイメージ */
    vector<vector<int>> test(3, vector<int>(10));
    
    /* 出席番号1番（インデックス0）の国語（インデックス0）の成績 */
    cout << test[0][0] << endl;
    
    /* 出席番号5番（インデックス4）の数学（インデックス2）の成績 */
    cout << test[2][4] << endl; // 0始まりに注意！
}

■ 使用例：九九の表を二次元配列に入れる

※配列は0から始まるので、1〜9の九九を入れる際も 0〜8 のループで回すのがC++の定石です。 

#include <bits/stdc++.h>
using namespace std;

int main(){
    // 9×9 の二次元配列を作成
    vector<vector<int>> kuku(9, vector<int>(9));
    
    // i, j はインデックスとして 0 から 8 まで回す
    for(int i = 0; i < 9; i++){
        for(int j = 0; j < 9; j++){
            // 実際の数字はインデックス+1 なので (i+1) * (j+1)
            kuku[i][j] = (i + 1) * (j + 1);
        }
    }
}

3. 文字（char）と ASCII コード

プログラミングは数値だけではなく、文字を扱うことができます。
文字ひとつだけ扱う char（文字）型です。 コンピュータの内部では、文字はすべて「数値」として管理されています。

■ 宣言と基本的な使い方

#include <bits/stdc++.h>
using namespace std;

int main(){
    char c = 'a'; // 文字はシングルクォーテーション '' で囲む
    cout << c << endl; // 'a' が出力される
}

■ 文字の正体：ASCII（アスキー）コード

C++において、char 型は内部的にはただの整数です。文字と数字の対応表を ASCII コード と呼びます。

文字 'A' 'B' ... 'Z' 'a' 'b' ... 'z' '0' '1'
ASCII値 65 66 ... 90 97 98 ... 122 48 49
💡 連続性のひみつと「変身魔法」
アルファベットや数字は、表の中で必ず連続して並んでいるのが重要なポイントです。
C++では、この特徴と変数の型を一時的にチェンジする技能を組み合わせることで、文字を自在に操ることができます。
型キャスト（型変換）の解説 ➔ 
#include <bits/stdc++.h>
using namespace std;

int main() {
    // 【文字 ➔ 数値】 'a' の背番号（ASCIIコード）を覗く
    cout << (int)'a' << endl;  // 97 と出力される

    // 【数値 ➔ 文字】 背番号 97 の文字を呼び出す
    cout << (char)97 << endl;  // 'a' と出力される
}

■ 実験：数値と文字を対応させてみよう

for ループを使って、1番から122番までの数値に対応する文字を順番に出力してみましょう。
(char)i と書くことで、整数を文字に変換（キャスト）して表示できます。 

#include <bits/stdc++.h>
using namespace std;

int main() {
    cout << "数値 : 文字" << endl;
    cout << "----------------" << endl;

    // 0〜31は画面に表示できない制御文字（改行など）が多い。
    // 実用的な文字が始まる32番から122番までを出力します
    for (int i = 0; i <= 122; i++) {
        // 数値 i を char 型に変換
        char c = (char)i; 
        
        cout << i << " : " << c << endl;
    }
}

■ 競プロ応用①：'c' - 'a' で「何番目のアルファベットか」を求める

文字同士で引き算をすると、ASCIIコードの差（引き算）が計算されます。
これを利用すると、'a' を基準（0番目）として、その文字がアルファベットの何番目かを一発で割り出せます。バケット（出現回数を数える配列）の実装で必須のテクニックです。 

#include <bits/stdc++.h>
using namespace std;

int main(){
    char c = 'f';
    
    // 'f' が 'a' から数えて何番目か（0始まり）
    int index = c - 'a'; 
    
    cout << index << endl; // 5 が出力される ('a'=0, 'b'=1, 'c'=2, 'd'=3, 'e'=4, 'f'=5)
}

■ 競プロ応用②：小文字 ⇔ 大文字の変換

ASCII 表を見ると、大文字の 'A'(65) と小文字の 'a'(97) の距離は 32 離れています。すべてのアルファベットでこの差は共通です。
小文字 - 32 = 大文字、 大文字 + 32 = 小文字
「小文字から 'a' を引いて 'A' を足す」ことで、大文字へ変換してあげましょう（逆も然り）。 

#include <bits/stdc++.h>
using namespace std;

int main(){
    char lowercase = 'm';
    
    // 小文字から大文字に変換
    char uppercase = lowercase - 'a' + 'A';
    
    cout << uppercase << endl; // 'M' が出力される
    
    // 【参考】数字の文字 '7' を、純粋な数値の 7 に変換する場合も同様
    char num_char = '7';
    int num = num_char - '0';
    cout << num << endl; // 数値の 7 が出力される
}

4. 文字列（string）

ここで文字の集まりである string 型を理解しましょう。
実は文字列も、「文字（char）」が連なった配列のようなものです。

■ 宣言と基本的な使い方

#include <bits/stdc++.h>
using namespace std;

int main(){
    string s = "Hello"; // sが変数
    cout << s.size() << endl; // 文字の長さ(5)を出力
    cout << s[0] << endl;     // "Hello"の最初の文字('H')を出力
}
💡 記号の打ち分け（重要）
文字列は ""（ダブルクォーテーション）、文字一つは ''（シングルクォーテーション）で囲みます。
※JIS配列キーボードなら "" = [Shift] + [2]、'' = [Shift] + [7] で出せます。

■ 応用例：特定の文字を数える

#include <bits/stdc++.h>
using namespace std;

int main(){
    string s = "C++ is a high-level, general-purpose programming language created by Danish computer scientist Bjarne Stroustrup.";
    int count = 0;
    
    /* sの中に 's' という文字がいくつあるかを数える */
    for(int i = 0; i < s.size(); i++){
        if(s[i] == 's'){
            count++;
        }
    }
    cout << count << endl;
}
