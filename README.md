# C++ 初心者講習会 〜ICPC 3完を目指す10講義〜

本リポジトリは、C++の基礎を固め、ICPC国内予選での「3完」およびAtCoder「茶色」レベルへの到達を目標とした初心者向け講習会の学習資料です。

ポインタ等の深淵には立ち入らず、競技プログラミングの実践で即座に役立つSTL（標準ライブラリ）の活用と、実装スピードを重視した構成になっています。

## 🎯 目標
- C++の基本文法およびSTLの習得
- ICPC国内予選 3完
- AtCoder 茶レベルの基礎力定着

## 📚 カリキュラム構成

| 講義 | テーマ | 主な内容 |
| :--- | :--- | :--- |
| **第00講** | CLIツール | ターミナル操作, コンパイル, コマンドTips |
| **第01講** | 入出力と計算 | `main`関数, `cin`/`cout`, 基本型, 型の限界 |
| **第02講** | 条件分岐 | `if`/`else`, 比較・論理演算子, 真偽値の扱い |
| **第03講** | 繰り返し処理 | `for`/`while`, 計算量の感覚, 全探索の基礎 |
| **第04講** | 関数と参照 | 関数の定義, 引数と戻り値, 参照渡し(`&`) |
| **第05講** | 配列と文字列 | `vector`, `string`, 多次元配列, 拡張for文 |
| **第06講** | STL(1) ソート | `sort`, `reverse`, `pair`, イテレータ基礎 |
| **第07講** | STL(2) 高速管理 | `set`, `map`, `queue`, `stack` |
| **第08講** | モダン文法・構造体 | 構造体(`struct`), 型推論(`auto`), 構造化束縛 |
| **第09講** | ICPC演習(1) | ICPC概要, 過去問解剖 |
| **第10講** | ICPC演習(2) | 総合考察, 全探索応用, 提出前チェック |
